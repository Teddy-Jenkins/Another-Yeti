Help! Everything is broken!

First, try restarting the game and Steam (if you’re using steam). If that doesn’t help, send an email to support@anotheryeti.com with a description of what’s going on, and a logfile. You can find the logfile in these locations:

Mac – ~/Library/Logs/Unity/Player.log
Windows – (install directory)\output_log.txt
Linux – ~/.config/unity3d/AnotherYeti/Metro\ Warp/Player.log

where ‘~’ is your home directory on Mac or Linux, and (install directory) is where the game is installed on Windows. For a Steam install, this will be C:\Program Files(x86)\Steam\steamapps\common\Metro Warp
